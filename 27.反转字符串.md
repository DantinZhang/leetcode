#### 错误写法：
错误的原因在于后置运算符的运用，这里你可能想的是先运算再+1，难道不对吗？  
其实这里先运算是先判断i<j，而不是先执行循环体，这里判断完后i+1，j+1，再进入循环体
```javascript
/**
 * @param {character[]} s
 * @return {void} Do not return anything, modify s in-place instead.
 */
var reverseString = function(s) {
    let i = 0;
    let j = s.length - 1;
    while(i++ < j--) {
        let temp = s[i];
        s[i] = s[j];
        s[j] = temp;
    }
};
```
#### 正确写法：
```javascript
/**
 * @param {character[]} s
 * @return {void} Do not return anything, modify s in-place instead.
 */
var reverseString = function(s) {
    let i = 0;
    let j = s.length - 1;
    while(i < j) {
        let temp = s[i];
        s[i] = s[j];
        s[j] = temp;
        i++;
        j--;
    }
};
```
